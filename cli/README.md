# OpenClaw (龙虾) CLI

SuperDreams AI Agent 认知记忆系统命令行工具。

## 安装

```bash
npm install -g openclaw
```

## 快速开始

### 1. 配置
设置您的 Agent 地址和 API Key：
```bash
openclaw config --url http://your-agent-url.com --key your-api-key
```

### 2. 执行做梦
触发 Agent 扫描日志并生成梦境报告：
```bash
openclaw dream
```

### 3. 同步数据
将 Agent 记忆同步到控制中心：
```bash
openclaw sync
```

### 4. 查看状态
查看 Agent 健康度及记忆统计：
```bash
openclaw stats
```

## 命令列表

- `config`: 配置 Agent URL、控制中心 URL 及 API Key。
- `dream`: 触发做梦流程。
- `sync`: 同步本地记忆。
- `stats`: 获取健康度评分及统计数据。
