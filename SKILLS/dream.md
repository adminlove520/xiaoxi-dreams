# 🌀 Dream — 小溪的认知记忆整合

## 何时使用

- 每天定时（cron 04:00）
- 哥哥说"整理记忆"、"做个梦"
- 日志堆积较多时
- "整理任务"、"检查待办"

## Dream 流程

```
触发 → 收集 → 整合 → 评估 → 报告
                 ↑
             告警检测
```

## Step 1: 收集 (Collect)

读取 `memory/YYYY-MM-DD.md`（最近7天未处理的日志）

提取内容：
- **decisions** — 决策
- **lessons** — 教训
- **people** — 人物
- **projects** — 项目
- **facts** — 事实
- **todos** — 待办

## Step 2: 整合 (Consolidate)

对比 `MEMORY.md`：
- 新内容 → 追加
- 已有内容 → 更新
- 重复 → 跳过

写入：
- `memory/procedures.md` — 工作流偏好
- `memory/episodes/` — 项目情景（如有）

标记已处理日志：`<!-- consolidated -->`

## Step 3: 评估 (Evaluate)

**重要性评分：**
```
importance = (base_weight × recency_factor × reference_boost) / 8.0
```

**健康度计算：**
```
health = (新鲜度×0.25 + 覆盖度×0.25 + 连贯度×0.2 + 效率×0.15 + 可达性×0.15) × 100
```

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         健康度告警阈值                                      │
└─────────────────────────────────────────────────────────────────────────────┘

  🔴 < 50: 严重告警
  ─────────────────────────────────────────────────────────────────────────
  • 立即推送给哥哥

  🟡 50-70: 轻度告警

```

## Dolt 命令

```bash
# 查看状态
xd.ps1 status

# 查看历史
xd.ps1 history

# 健康度趋势
xd.ps1 health

# 记忆统计
xd.ps1 stats

# 备份
.\backup.ps1          # 增量备份
.\backup.ps1 -Full   # 全量备份
.\backup.ps1 -List    # 列出备份

# 提交变更
xd.ps1 commit

# 推送到远程
xd.ps1 push
```

## Beads 任务追踪

```bash
bd create "任务名" -p 0    # P0 最高优先级
bd ready                   # 查看可执行任务
bd update <id> --claim    # 领取任务
bd close <id> --reason ""  # 完成
```

## 输出规则

- 推送到当前聊天
- 报告存入 `memory/dream-log.md`
- 记录存入 Dolt 数据库
- 严重告警立即通知哥哥

## 安全规则

1. 永不删除日志文件 — 只标记 `<!-- consolidated -->`
2. 永不移除 `⚠️ PERMANENT` 标记的条目
3. 大变更前先备份
4. MEMORY.md 上限 200 行

## 智能跳过逻辑

```
近 7 天无新日志？ → ✅ 是 → 快速退出
健康度 > 90？     → ✅ 是 → 轻量检查
上次 Dream < 6小时？→ ✅ 是 → 跳过
```
