# ⚡ 性能优化

> xiaoxi-dreams 性能优化指南

## 当前瓶颈分析

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         性能瓶颈                                        │
└─────────────────────────────────────────────────────────────────────────────┘

1. 全量扫描 (最耗时)
   ├─ 每次 Dream 扫描 7 天日志
   ├─ 重复读取未变更的文件
   └─ 没有缓存机制

2. Context 膨胀
   ├─ 加载 MEMORY.md 全文
   ├─ 加载所有 Dolt 查询结果
   └─ 没有分层加载

3. Dolt 查询
   ├─ 每次查询重新连接
   ├─ 没有结果缓存
   └─ 大表扫描慢

4. Token 消耗
   ├─ 每次报告生成完整 prompt
   ├─ 没有增量上下文
   └─ 重复的通用内容
```

## 优化策略

### 1. 增量扫描

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         增量扫描策略                                     │
└─────────────────────────────────────────────────────────────────────────────┘

之前: 全量扫描
┌─────────────────────────────────────────────────────────────────────┐
│  Day 1 │ Day 2 │ Day 3 │ Day 4 │ Day 5 │ Day 6 │ Day 7 │
│   ✓    │   ✓    │   ✓    │   ✓    │   ✓    │   ✓    │   ✓    │ ← 7天全部读取
└─────────────────────────────────────────────────────────────────────┘

之后: 增量扫描
┌─────────────────────────────────────────────────────────────────────┐
│  Day 1 │ Day 2 │ Day 3 │ Day 4 │ Day 5 │ Day 6 │ Day 7 │
│   -    │   -    │   ✓    │   ✓    │   ✓    │   ✓    │   ✓    │ ← 只读新的
│  (已处理)    (已处理)   (新增)                              │
└─────────────────────────────────────────────────────────────────────┘

实现:
1. 记录上次扫描的日期
2. 只扫描该日期之后的文件
3. 已处理文件标记 <!-- consolidated -->
```

### 2. 分层加载

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         分层加载策略                                     │
└─────────────────────────────────────────────────────────────────────────────┘

Layer 1: 索引 (MEMORY.md)
├─ 大小: ~10KB
├─ 内容: ID、标题、摘要
└─ 用途: 快速导航

Layer 2: 元数据 (Dolt 查询)
├─ 大小: ~5KB (JSON)
├─ 内容: 统计、健康度
└─ 用途: 概览

Layer 3: 详细内容 (按需)
├─ 大小: 可变
├─ 内容: 完整记忆内容
└─ 用途: 需要时再加载
```

### 3. 缓存机制

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         缓存策略                                         │
└─────────────────────────────────────────────────────────────────────────────┘

缓存层级:
┌─────────────────────────────────────────────────────────────────────┐
│  L1: 内存缓存 (进程内)                                          │
│  ├─ 健康度统计: 5 分钟 TTL                                      │
│  ├─ 记忆列表: 10 分钟 TTL                                      │
│  └─ 配置: 无过期                                               │
├─────────────────────────────────────────────────────────────────────┤
│  L2: 文件缓存 (disk)                                           │
│  ├─ 索引快照: 1 小时 TTL                                      │
│  ├─ 统计缓存: 30 分钟 TTL                                     │
│  └─ 清理: 每天 04:00                                          │
├─────────────────────────────────────────────────────────────────────┤
│  L3: Dolt 缓存                                                 │
│  ├─ 连接池: 复用连接                                           │
│  ├─ 查询结果: 5 分钟 TTL                                       │
│  └─ Prepared statements                                        │
└─────────────────────────────────────────────────────────────────────┘
```

### 4. Prompt 优化

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         Prompt 优化                                     │
└─────────────────────────────────────────────────────────────────────────────┘

之前: 完整 Prompt
┌─────────────────────────────────────────────────────────────────────┐
│ System: [完整 SKILL 说明 + 所有规则 + 历史]                    │
│ User: [生成报告]                                                  │
│                                                                     │
│ Tokens: ~2000 (每次)                                              │
└─────────────────────────────────────────────────────────────────────┘

之后: 增量 Prompt
┌─────────────────────────────────────────────────────────────────────┐
│ System: [基础 SKILL 说明]                                        │
│ Context: [本次需要的数据]                                        │
│ User: [本次任务]                                                  │
│                                                                     │
│ Tokens: ~800 (减少 60%)                                         │
└─────────────────────────────────────────────────────────────────────┘
```
