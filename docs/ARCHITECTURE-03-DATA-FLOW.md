## 数据一致性保证

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         数据一致性机制                                      │
└─────────────────────────────────────────────────────────────────────────────┘

  1. Content Hash 去重
  ┌───────────────────────────────────────────────────────────────────────┐
  │  INSERT INTO memory_entries                                          │
  │  WHERE content_hash NOT IN (SELECT content_hash FROM ...)            │
  └───────────────────────────────────────────────────────────────────────┘

  2. 事务保证 (Dolt Auto-Commit)
  ┌───────────────────────────────────────────────────────────────────────┐
  │  每条写入 → 自动 Dolt Commit → 可回滚                                 │
  └───────────────────────────────────────────────────────────────────────┘

  3. 备份策略
  ┌───────────────────────────────────────────────────────────────────────┐
  │  daily:  xd.ps1 backup (压缩 .beads/)                              │
  │  push:   xd.ps1 push (推送到 Git remote)                            │
  │  restore: xd.ps1 restore <backup-file>                               │
  └───────────────────────────────────────────────────────────────────────┘
```
