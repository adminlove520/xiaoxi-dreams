# 🧠 记忆层级结构

## 5 层记忆架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         5 层记忆架构                                      │
└─────────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  Layer 5: Index (索引层)                                               │
  │  ─────────────────────────────────────────────────────────────────────│
  │                                                                         │
  │  文件: memory/index.json                                                │
  │                                                                         │
  │  内容:                                                                  │
  │  ├─ version           版本号                                            │
  │  ├─ lastDream         上次做梦时间                                      │
  │  ├─ health            健康度评分                                        │
  │  ├─ dreamCount        做梦次数                                          │
  │  └─ statistics        统计信息                                          │
  │                                                                         │
  │  用途: 只做索引导航，不进 context                                      │
  │  访问: 按需加载，不是每次都读                                           │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘
                                    ▲
                                    │ 路由查询
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  Layer 4: Procedural (程序记忆)                                         │
  │  ─────────────────────────────────────────────────────────────────────│
  │                                                                         │
  │  文件: memory/procedures.md                                             │
  │                                                                         │
  │  内容:                                                                  │
  │  ├─ 工作流      执行过的工作流程                                         │
  │  ├─ 偏好        哥哥的偏好和习惯                                        │
  │  ├─ 工具用法   工具使用的最佳实践                                       │
  │  └─ 习惯       常见的操作模式                                            │
  │                                                                         │
  │  更新: 当工作方式改变时                                                  │
  │  示例:                                                                │
  │  ## openclaw config set                                               │
  │  - 安全修改配置的方法 (比手动编辑更安全)                               │
  │  - 自动创建 .bak 备份                                                │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘
                                    ▲
                                    │ 路由查询
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  Layer 3: Long-term (长期记忆)                                          │
  │  ─────────────────────────────────────────────────────────────────────│
  │                                                                         │
  │  文件: MEMORY.md                                                       │
  │                                                                         │
  │  内容:                                                                  │
  │  ├─ 事实        已验证的知识                                            │
  │  ├─ 决策        重要决策及理由                                         │
  │  ├─ 人物        人物关系 (memory/people/)                              │
  │  └─ 战略        长期目标和方向                                          │
  │                                                                         │
  │  限制:                                                                  │
  │  ├─ 上限 200 行                                                        │
  │  ├─ 每条目 ~150 chars                                                  │
  │  └─ 详细内容存 topic files                                              │
  │                                                                         │
  │  格式:                                                                  │
  │  ## [mem_xxx] 标题                                                    │
  │  - 内容摘要                                                            │
  │  - 来源: YYYY-MM-DD                                                    │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘
                                    ▲
                                    │ 路由查询
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  Layer 2: Episodic (情景记忆)                                           │
  │  ─────────────────────────────────────────────────────────────────────│
  │                                                                         │
  │  文件: memory/episodes/*.md                                             │
  │                                                                         │
  │  内容:                                                                  │
  │  ├─ 项目叙事   项目的完整故事线                                         │
  │  ├─ 事件时间线  重大事件的发展过程                                      │
  │  └─ 跨会话上下文  跨会话的重要背景                                      │
  │                                                                         │
  │  命名: YYYY-MM-DD-project-name.md                                      │
  │                                                                         │
  │  示例:                                                                  │
  │  2026-04-02-xiaoxi-dreams-launch.md                                   │
  │  - 项目启动的背景                                                       │
  │  - 关键决策点                                                           │
  │  - 结果和教训                                                           │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘
                                    ▲
                                    │ 提取
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  Layer 1: Work (工作记忆)                                               │
  │  ─────────────────────────────────────────────────────────────────────│
  │                                                                         │
  │  文件: memory/YYYY-MM-DD.md                                             │
  │                                                                         │
  │  内容:                                                                  │
  │  ├─ 原始日志    每小时的时间戳日志                                       │
  │  ├─ 决策记录   ## Decision: xxx                                        │
  │  ├─ 教训记录   ## Lesson: xxx                                          │
  │  └─ 待办        ## TODO: xxx                                           │
  │                                                                         │
  │  生命周期:                                                              │
  │  ├─ 7 天内     保留原始文件                                            │
  │  ├─ 7 天后     标记 consolidated                                     │
  │  └─ 90 天后    可归档 (非永久)                                        │
  │                                                                         │
  │  标记:                                                                  │
  │  - <!-- consolidated -->  已整合                                        │
  │  - <!-- archived -->      已归档                                        │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘
```

## 记忆流向

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         记忆流转方向                                        │
└─────────────────────────────────────────────────────────────────────────────┘

                              ┌─────────────────┐
                              │   用户输入       │
                              │  (哥哥对话)      │
                              └────────┬────────┘
                                       │
                                       ▼
                    ┌──────────────────────────────────────┐
                    │        Layer 1: Work                 │
                    │        每日日志写入                    │
                    │        memory/YYYY-MM-DD.md           │
                    └──────────────────┬───────────────────┘
                                       │
                                       │ Dream 时提取
                                       ▼
                    ┌──────────────────────────────────────┐
                    │        Layer 2: Episodic             │
                    │        情景整合 (如有重大事件)         │
                    │        memory/episodes/               │
                    └──────────────────┬───────────────────┘
                                       │
                                       │ 定期整合
                                       ▼
                    ┌──────────────────────────────────────┐
                    │        Layer 3: Long-term            │
                    │        长期记忆                      │
                    │        MEMORY.md                     │
                    └──────────────────┬───────────────────┘
                                       │
                                       │ 按需更新
                                       ▼
                    ┌──────────────────────────────────────┐
                    │        Layer 4: Procedural          │
                    │        程序记忆                       │
                    │        memory/procedures.md           │
                    └──────────────────┬───────────────────┘
                                       │
                                       │ 索引更新
                                       ▼
                    ┌──────────────────────────────────────┐
                    │        Layer 5: Index                 │
                    │        索引                           │
                    │        memory/index.json              │
                    └──────────────────────────────────────┘

  访问方向 (反向):
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  查询请求 ──► Index ──► Long-term ──► Episodic ──► Work              │
  │              (快速导航)   (按需加载)   (详情)      (原始)            │
  └─────────────────────────────────────────────────────────────────────────┘
```

## 重要性分级

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         记忆重要性分级                                      │
└─────────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  P0: ⚠️ PERMANENT (永久)                                               │
  │  ─────────────────────────────────────────────────────────────────────│
  │  ├─ 永远不会删除                                                        │
  │  ├─ 核心身份、价值观                                                   │
  │  └─ 示例: SOUL.md 的核心原则                                           │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  P1: 🔥 HIGH (高)                                                       │
  │  ─────────────────────────────────────────────────────────────────────│
  │  ├─ 重要性 8-10                                                        │
  │  ├─ 长期有效的知识                                                      │
  │  └─ 示例: 关键技术决策、核心项目                                         │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  P2: 📌 PIN (置顶)                                                      │
  │  ─────────────────────────────────────────────────────────────────────│
  │  ├─ 重要性 6-7                                                        │
  │  ├─ 当前正在使用的流程                                                  │
  │  └─ 示例: 当前项目的工作流                                              │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  P3: 普通 (常规)                                                        │
  │  ─────────────────────────────────────────────────────────────────────│
  │  ├─ 重要性 4-5                                                        │
  │  ├─ 一般性知识                                                        │
  │  └─ 示例: 日常操作记录                                                 │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  P4: 可归档 (低价值)                                                   │
  │  ─────────────────────────────────────────────────────────────────────│
  │  ├─ 重要性 < 3                                                         │
  │  ├─ 90 天无引用                                                        │
  │  └─ 处理: 归档而非删除                                                  │
  └─────────────────────────────────────────────────────────────────────────┘
```

## 遗忘曲线

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         遗忘曲线                                          │
└─────────────────────────────────────────────────────────────────────────────┘

  引用频率
     ▲
     │    ╭─────────────── 永久记忆 (PERMANENT)
     │   ╱│ P1: HIGH
  0.8├──╯ │
     │   ╲│ P2: PIN
  0.6├────╯
     │     ╲
  0.4├──────╲
     │       ╲
  0.2├────────╲─────── P3: 普通
     │         ╲
  0.0├──────────╲───────────────── P4: 可归档
     └───────────▼─────────────────────────► 时间
          0    30    60    90    180    365
                   天数

  归档条件:
  ┌───────────────────────────────────────────────────────────────────────┐
  │  > 90 天未引用 AND importance < 0.3 AND 非 PERMANENT                │
  └───────────────────────────────────────────────────────────────────────┘
```
