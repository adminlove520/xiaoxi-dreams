# 🌙 Dream 流程图

## 完整流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           Dream 执行流程                                      │
└─────────────────────────────────────────────────────────────────────────────┘

                                    ┌─────────────────┐
                                    │  触发 Dream     │
                                    │  - Cron 04:00   │
                                    │  - 手动触发     │
                                    └────────┬────────┘
                                             │
                                             ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 1: 收集 (Collect)                                                    │
│ ─────────────────────────────────────────────────────────────────────────│
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │  扫描 memory/ 目录                                                 │      │
│   │  条件：最近 7 天内，未标记 consolidated                            │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                                                             │
│   提取内容：                                                               │
│   ├─ Decisions    (决策)    → 路由到 MEMORY.md                          │
│   ├─ Lessons      (教训)    → 路由到 memory/lessons/                    │
│   ├─ People       (人物)    → 路由到 memory/people/                     │
│   ├─ Procedures   (流程)    → 路由到 memory/procedures.md               │
│   ├─ Projects     (项目)     → 路由到 memory/episodes/                   │
│   └─ Todos        (待办)    → 路由到 ACTIVE-TASK.md                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                             │
                                             ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 2: 整合 (Consolidate)                                               │
│ ─────────────────────────────────────────────────────────────────────────│
│                                                                             │
│   对比 MEMORY.md，检查重复：                                               │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │  重复检测                                                          │      │
│   │  ├─ content_hash 匹配 → 跳过 (已有)                              │      │
│   │  ├─ 内容相似度 > 80%  → 合并 (更新)                              │      │
│   │  └─ 新内容          → 追加 (新增)                                 │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                                                             │
│   写入 Dolt：                                                             │
│   ├─ INSERT INTO dream_sessions (...)     记录本次 Dream                  │
│   └─ INSERT INTO memory_entries (...)     记忆条目                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                             │
                                             ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 3: 评估 (Evaluate)                                                  │
│ ─────────────────────────────────────────────────────────────────────────│
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │  重要性评分                                                        │      │
│   │                                                                 │      │
│   │  importance = (base_weight × recency_factor × reference_boost) │      │
│   │                                                                 │      │
│   │  ├─ ⚠️ PERMANENT  → 权重 1.0 (固定)                            │      │
│   │  ├─ 🔥 HIGH        → 权重 2.0                                  │      │
│   │  ├─ 📌 PIN         → 权重 1.5                                  │      │
│   │  └─ 普通条目       → 权重 1.0                                  │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │  健康度计算                                                        │      │
│   │                                                                 │      │
│   │  health = (新鲜度×0.25 + 覆盖度×0.25 + 连贯度×0.2            │      │
│   │             + 效率×0.15 + 可达性×0.15) × 100                   │      │
│   │                                                                 │      │
│   │  ├─ 新鲜度    → 最近30天被引用占比                              │      │
│   │  ├─ 覆盖度    → 最近14天有更新类别占比                         │      │
│   │  ├─ 连贯度    → 有关联条目占比                                 │      │
│   │  ├─ 效率      → MEMORY.md 精简程度                             │      │
│   │  └─ 可达性    → 记忆图谱连通程度                               │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │  洞察生成                                                          │      │
│   │  分析本次 Dream 的关键发现和建议                                  │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                             │
                                             ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 4: 报告 (Report)                                                   │
│ ─────────────────────────────────────────────────────────────────────────│
│                                                                             │
│   输出到：                                                               │
│   ├─ Telegram Bot     → 推送给哥哥 (主要)                               │
│   ├─ memory/dream-log.md  → 存档                                       │
│   └─ Dolt            → 记录到 health_metrics                            │
│                                                                             │
│   报告内容：                                                             │
│   ├─ 📊 统计    → 扫描文件数、新增、更新                               │
│   ├─ 🧠 健康度  → 综合评分 + 变化趋势                                  │
│   ├─ 🔮 洞察    → 关键发现                                            │
│   ├─ 📝 变更    → 新增/更新的记忆条目                                  │
│   └─ 💡 建议    → 后续行动建议                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                             │
                                             ▼
                                      ┌─────────────────┐
                                      │   Dream 完成     │
                                      │   标记日志 consolidated  │
                                      └─────────────────┘
```

## 智能跳过条件

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           智能跳过检查                                      │
└─────────────────────────────────────────────────────────────────────────────┘

开始 Dream
    │
    ├─ 近 7 天无新日志？ ──→ ✅ 是 ──→ 快速退出 (节省 token)
    │
    ├─ 健康度 > 90？ ──────→ ✅ 是 ──→ 轻量检查
    │
    ├─ 上次 Dream < 6小时？ → ✅ 是 ──→ 跳过
    │
    └─ 所有检查通过 ──────────→ 执行完整 Dream 流程
```
