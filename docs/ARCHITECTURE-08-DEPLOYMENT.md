## 安装流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           安装流程                                          │
└─────────────────────────────────────────────────────────────────────────────┘

  1. 克隆仓库
  ─────────────────────────────────────────────────────────────────────────
  $ git clone https://github.com/adminlove520/xiaoxi-dreams.git
  $ cd xiaoxi-dreams

  2. 运行安装脚本
  ─────────────────────────────────────────────────────────────────────────
  # Windows
  .\scripts\setup.ps1

  # Linux/macOS
  ./scripts/setup.sh

  3. 安装依赖
  ─────────────────────────────────────────────────────────────────────────
  $ go install github.com/dolthub/dolt/cmd/dolt@latest
  $ go install github.com/steveyegge/beads/cmd/bd@latest

  4. 初始化 Dolt
  ─────────────────────────────────────────────────────────────────────────
  $ dolt init
  $ bd init --server

  5. 重启 Gateway
  ─────────────────────────────────────────────────────────────────────────
  $ openclaw gateway restart

  6. 验证安装
  ─────────────────────────────────────────────────────────────────────────
  $ "做个梦"  # 手动触发 Dream
```

## 备份策略

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           备份策略                                          │
└─────────────────────────────────────────────────────────────────────────────┘

  本地备份 (xd.ps1 backup)
  ─────────────────────────────────────────────────────────────────────────
  备份内容: .beads/ 目录 (Dolt 数据库)
  备份位置: backups/backup-YYYYMMDD-HHMMSS.tar.gz
  保留策略: 最近 7 个备份

  Git Remote (自动)
  ─────────────────────────────────────────────────────────────────────────
  $ dolt add . && dolt commit -m "message"
  $ dolt push origin main

  ⚠️ 注意: .beads/dolt/ 已被 .gitignore 忽略
           只有配置文件被版本控制

  远程备份 (S3) [待实现]
  ─────────────────────────────────────────────────────────────────────────
  $ xd.ps1 backup --remote s3://bucket/backups/

  恢复
  ─────────────────────────────────────────────────────────────────────────
  $ xd.ps1 restore backups/backup-20260402.tar.gz
```

## 故障排查

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           故障排查                                          │
└─────────────────────────────────────────────────────────────────────────────┘

  问题: Dream 不执行
  ─────────────────────────────────────────────────────────────────────────
  1. 检查 cron 状态: openclaw cron list
  2. 检查 Dolt Server: curl http://127.0.0.1:57823
  3. 手动触发: "做个梦"

  问题: Dolt Server 连接失败
  ─────────────────────────────────────────────────────────────────────────
  1. 检查 PID: cat .beads/dolt-server.pid
  2. 检查端口: cat .beads/dolt-server.port
  3. 重启 Server: bd dolt start

  问题: 数据不同步
  ─────────────────────────────────────────────────────────────────────────
  1. 检查状态: dolt status
  2. 提交本地: dolt add . && dolt commit -m "fix"
  3. 推送: dolt push origin main
  4. 拉取: dolt pull origin main

  问题: Beads 命令失败
  ─────────────────────────────────────────────────────────────────────────
  1. 检查 bd 版本: bd --version
  2. 重新初始化: bd init --server
  3. 检查配置: cat .beads/config.yaml
```
