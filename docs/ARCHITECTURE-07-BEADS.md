## 跨系统任务创建

```bash
# 1. Dream 发现新需求
"需要在下次 Dream 中添加健康度告警功能"

# 2. 创建 Beads 任务
bd create "添加健康度告警" -p 1 -t feature

# 3. 自动记录到 memory_entries
# (由 xd.ps1 sync 命令同步)

# 4. 下次 Dream 时检查
bd ready  # 查看所有可执行任务

# 5. 领取并执行
bd update <task_id> --claim
# ... work ...
bd close <task_id> --reason "已完成"
```

## 协作命令速查

| 命令 | 说明 | xiaoxi-dreams 集成 |
|------|------|---------------------|
| `bd ready` | 查看可执行任务 | ✅ 自动同步 |
| `bd create "标题" -p 0` | 创建任务 | ✅ 记录到 memory |
| `bd update <id> --claim` | 领取任务 | ✅ 记录变更 |
| `bd close <id> --reason ""` | 完成任务 | ✅ 同步到 dream |
| `bd dep add <child> <parent>` | 添加依赖 | ❌ 待实现 |
| `bd dolt push` | 推送变更 | ✅ xd.ps1 push |
| `bd dolt pull` | 拉取变更 | ✅ xd.ps1 pull |
